# LCS

[9251 LCS](https://www.acmicpc.net/problem/9251)

LCS(Longest Common Subsequence, 최장 공통 부분 수열)
DP

한 글자씩 비교하면서 두 글자가 같을 때

dp[i][j] = dp[i - 1][j - 1] + 1;

다를 때

dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
